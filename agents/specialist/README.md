# Specialist Agents

Specialist files define narrow Codex roles for FIXICS work.

## Roles

- `sqf-agent.md`: SQF functions, mission behavior, and runtime scripting.
- `config-agent.md`: `config.cpp`, `CfgPatches`, `CfgFunctions`, and addon metadata.
- `qa-agent.md`: HEMTT validation and manual Arma smoke testing.

Use one primary specialist at a time unless a task clearly crosses boundaries.
