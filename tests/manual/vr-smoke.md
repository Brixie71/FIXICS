# VR Smoke Procedure

## Command

```powershell
hemtt launch vr
```

or:

```powershell
.\tools\launch-vr.ps1
```

## Procedure

1. Launch the VR mission.
2. Wait for the player unit to initialize.
3. Confirm the FIXICS hint appears.
4. Confirm the FIXICS system chat line appears.
5. Confirm dynamic text appears near the upper screen area.
6. Check for script error popups.

## When Required

Run this after visible gameplay, UI, mission, or launch-profile changes.
